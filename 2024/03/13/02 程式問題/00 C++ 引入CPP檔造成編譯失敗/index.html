C++ 引入CPP檔造成編譯失敗

[!example]+ EX
include "cpp", duplicate difinition

程式

引入CPP檔不安全，上述例子因為main引入了calculator，link的時候main有定義、calculator.cpp也有定義，造成重複定義

報錯

解決:

[!success] 解決

  1. 使用 .h + .cpp
  2. 全部放入 .h

1. .h + .cpp

2. .h
